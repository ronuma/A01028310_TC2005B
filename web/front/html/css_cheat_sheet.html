Rodrigo's CSS Cheat Sheet

border

Coloca el borde de un elemento, como lo que sucede aquí.

border-left, border-right, border-top, border-bottom

Similar al border, solo que con estas propiedades es posible especificar solo algunos de los bordes del elemento.

border-radius

Esta propiedad permite alterar qué tan redondas son las esquinas del contenido, permitiendo suavizar el contenido, que se vea más amigable.

Es posible suavizar diferentes esquinas del contenido, independientemente.

Mira nomás

background-image

Permite insertar imágenes de fondo en un elemento, como en este.

color

Con esta propiedad se pueden asignar distintos colores a distintos elementos.

font

Esta propiedad permite establecer todas las características de la tipografía, como la familia, el estilo, el tamaño, el peso, entre otras. Sin embargo, parece que no está funcionando ahorita, aunque lo estoy usando tal cual aparece en la documentación.

font-family

Permite específicamente establecer la familia de fuente, o sea, la tipografía.

font-style, font-weight

Style sirve para el estilo de la fuente, o sea cursiva por ejemplo, y font weight para el peso, o sea, qué tan "gorda" es.

width, height

Esto es para determinar qué tan ancho o alto es un elemento, como estos divs:

Long Div
Wide Div

list-style

Según la documentación de MDN Web Docs, esta propiedad permite definir varias propiedades a la vez, sobre una lista, que son:

    Lista
  • list-style-type: definir qué tipo de bullet se utiliza, como este cuadrado
  • list-style-image: puedo agregar una imagen al elemento de la lista, al mismo tiempo que altero también el tipo del bullet
  • list-style-position: permite alterar la posicion en la que se encuentra la lista respecto a su padre.

margin

Permite establecer el margen del elemento, de forma estática y dinámica, para sus cuatro lados, en lugar de tener que usar individualmente margin-bottom, margin-top, etc, que son los que se usan para cada lado específico.

Márgenes para cada lado.

padding

Este es parecido a margin, solo que este tiene que ver con el espacio que hay entre el contenedor y su hijo, por ejemplo, veamos el div de ejemplo de margen pero ahora con padding en lugar de margen.

Y vemos como ahora el contenido es lo que tiene espacio alrededor, no el elemento como tal.

overflow

Permite indicar qué hacer cuando el contenido excede el espacio que debería ocupar.

Aquí se indica que el overflow tiene que ser scrolleable, como se puede ver.

display

Esta propiedad determina cómo se tienen que mostrar los elementos (bloque o en línea) y cómo se muestran sus hijos (como flexible o grid).

position

Esta propiedad determina cómo se posicionan los elementos, de diferentes maneras, y puede ser útil, por ejemplo, para colocar elementos por encima de otros, utilizando la posición absoluta, mientras que su padre utiliza posición relativa.
