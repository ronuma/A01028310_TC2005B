Rodrigo's CSS Cheat Sheet

border

Coloca el borde de un elemento, como lo que sucede aquí.

border-left, border-right, border-top, border-bottom

Similar al border, solo que con estas propiedades es posible especificar solo algunos de los bordes del elemento.

border-radius

Esta propiedad permite alterar qué tan redondas son las esquinas del contenido, permitiendo suavizar el contenido, que se vea más amigable.

Es posible suavizar diferentes esquinas del contenido, independientemente.

Mira nomás

background-image

Permite insertar imágenes de fondo en un elemento, como en este.

color

Con esta propiedad se pueden asignar distintos colores a distintos elementos.

font

Esta propiedad permite establecer todas las características de la tipografía, como la familia, el estilo, el tamaño, el peso, entre otras.
